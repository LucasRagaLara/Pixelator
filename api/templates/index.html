Sobre el proyecto

Este sistema forma parte de un microservicio en Docker que incluye detección de caras, clasificación, pixelado y almacenamiento. Diseñado por Lucas Raga como solución para anonimización de imágenes en contextos donde se requiere preservar la privacidad.
